## 募集中のポジション

### Webデベロッパー

Nova VNDBのウェブサイトを構築・維持するバックエンド開発者およびフロントエンド開発者を募集しています。

バックエンド開発者には、以下の技術に精通していることを求めます：

- Rust
- axum
- sqlx
- tokio
- tower
- PostgreSQL
- RabbitMQ
- Redis
- S3

フロントエンド開発者には、以下の技術に精通していることを求めます：

- Svelte
- SvelteKit
- Astro
- Tailwind CSS
- TypeScript

開発はRustとTypeScriptを中心に進めるため、両言語に慣れている方を募集しています。コードベースはオープンソースであり、GitHubで公開予定です。

中級以上の英語力が必要です。

選考は以下のステップで行います：

1. メールにて応募内容を確認します。
2. フロントエンド開発者の場合、オープンソースへの貢献実績や過去のプロジェクトを確認します。作品へのリンクを最低1件提出してください。
3. バックエンド開発者の場合、コーディングスキルの確認として、C・C++・Rustのいずれかを使用し、30分以内に中程度の難易度のLeetCode問題を解いていただきます。AIの利用は可能です。
4. 上記ステップの結果に応じて、スキルやプロジェクトへの理解を深く評価するための音声面接を別途実施する場合があります。面接は英語または日本語で行い、コミュニケーション能力とプロジェクトへの洞察力を評価します。

### 国際化スタッフ

VNDBプロジェクトは主に英語で運営されているため、英語に加えて少なくとも1つの言語に堪能で、VNDBプロジェクトへの深い理解をお持ちの方を募集しています。

上級以上の英語力が必要です。英語が母国語でない方は、以下の基準を参考にしてください：

- TOEFL：80点以上
- IELTS：6.5以上
- TOEIC：900点以上

ビジュアルノベルの多くは日本語であるため、日本語に堪能で日本語・日本文化への深い理解をお持ちの方を特に歓迎します。

日本語が母国語でない方は、以下の試験での実力証明をお勧めします：

- JLPT：N1
- BJT：J2以上

プライバシーへの配慮から、試験のスコアや証明書の提出を求めることはありません。代わりに、英語および/または日本語で短い筆記面接を実施し、スキルを評価します。

中国語・スペイン語など他の言語も歓迎します。いずれの言語においても、英語または日本語のいずれかで上級以上の能力が必要です。

## よくある質問

### フルタイムのポジションですか？

いいえ、ボランティアポジションです。パートタイム・フルタイムのどちらでも、ご都合に合わせてご参加いただけます。

### 報酬はありますか？

いいえ、ボランティアポジションのため報酬はありません。パートタイム・フルタイムのどちらでも、ご都合に合わせてご参加いただけます。

### デベロッパーポジションに応募しなくてもコードベースに貢献できますか？

必ずしも応募が必要なわけではありません。プロジェクトがベータ段階に達した後は、全員にプルリクエストの権限を開放する予定です。それまでの間は、採用されたデベロッパーのみがコードベースに直接貢献できます。

デベロッパーポジションに応募された場合、アカウントに「サイトデベロッパー」バッジが付与されます。

### 言語テストをまだ受けていませんが、応募できますか？

はい、応募できます。上記のテストは必須ではありませんが、目指すべき基準としてご参考にしてください。

### このプロジェクトにどのくらいの時間が必要ですか？

応募するポジションによって異なります。

デベロッパーポジションの場合、初期段階では一定のコーディング時間を確保していただく必要があります。厳格な時間要件はなく、タスクは自由に選んでいただけますが、担当するタスクは期限内に完了してください。プロジェクトがベータ段階に達した後は、必要なコミット量はほぼゼロになります。

国際化スタッフポジションの場合、コンテンツやUI文字列の翻訳にある程度の時間を費やしていただきます。初期段階では、お住まいのタイムゾーンに合わせてオンラインでご対応いただく必要があります。プロジェクトがアルファ段階に達した後は、ご自身のペースで作業いただけます。

コミュニティモデレーターの場合、少なくとも週2回のコミュニティ確認をお願いします。

### 結果はいつ通知されますか？

各ステップの結果は、東京時間で1営業日以内にお知らせします。
